# Pet-Shop-Management-Project
This is a C#-based application designed to manage and automate pet store operations. It provides a secure admin login to ensure only authorized users can access and control the system. It helps manage customer details, pets, products, and users information in an organized way. The system allows administrators to store and retrieve user data, track sales, maintain inventory, and handle customer interactions efficiently. By centralizing information such as user profiles, pet records, and transation details, the system reduces manual work and improves accuracy. Overall, it provides a simple yet effective solution for managing day-to-day activities in a pet shop.
